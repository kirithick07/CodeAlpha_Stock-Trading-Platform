# CodeAlpha_Stock-Trading-Platform

Stock Trading System – Java

Abstract

This project is a simple Stock Trading System developed using Java and Object-Oriented Programming concepts. The system allows users to view stock prices, buy stocks, sell stocks, and manage their portfolio through a console-based interface.

Features

- View available stocks
- Buy stocks
- Sell stocks
- Display portfolio details
- Track account balance

Technologies Used

- Java
- OOP Concepts
- HashMap Collection
- Console Application

Classes Used

Stock.java

Stores stock name and stock price.

Portfolio.java

Manages:

- Balance
- Buy operation
- Sell operation
- Portfolio display

StockTrading.java

Main class that handles menu operations.

Algorithm

1. Start program.
2. Create stock objects.
3. Create portfolio object.
4. Display menu options.
5. Read user choice.
6. Perform Buy / Sell / View / Portfolio operation.
7. Repeat until Exit.
8. End program.

Output

Users can perform stock trading operations and view updated portfolio information.

Conclusion

The Stock Trading System demonstrates practical implementation of Java OOP concepts including classes, objects, encapsulation, and collections.
